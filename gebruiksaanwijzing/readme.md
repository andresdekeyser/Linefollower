# Gebruiksaanwijzing

### opladen / vervangen batterijen
De robot werkt op twee 18650 Li-ion batterijen. Gebruik steeds een geschikte externe lader om deze op te laden. Voor het vervangen trek je de velcrostrip voorzichtig los en open je het klepje van de batterijhouder. De batterijen kunnen vervolgens eenvoudig uitgenomen en vervangen worden.

### draadloze communicatie
de draadloze communicatie start wanneer het lichtje brand van de nano esp32 en het gaat altijd van zelf je moet dus zelf niks doen 

#### verbinding maken
verbind je toestel met de wifi van het robotje en vul het wachtwoord in. Dan je op internet surfen naar het ip adres van de robot (192.168.4.1).dan komt u uit op de webpagina  

### kalibratie
kalibratie moet niet de sensor ziet wanneer er geen zwarte lijn is door zijn hoge waarden. maak wel dat er genoeg licht is op de zwarte lijn 

### settings
de standaart werken instellingen voor Kp , Kd , basspeed , maxspeed zijn 
Kp = 2
Kd = 140
basspeed = 90
maxspeed = 130

### start/stop button
je kan via de webpagina de robot starten/stoppen of via de knop op de robot zelf
